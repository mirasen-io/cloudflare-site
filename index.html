Current product direction

Mirasen Chess Lore

Chess Lore is the current main product direction inside Mirasen: a workflow-first system for structured chess learning, starting with trainer workflows and expanding into broader chess learning over time.

The current direction focuses on helping trainers assign, organize, review, and reinforce chess learning with more structure. The same learning logic also supports self-directed learners who want clearer review, stronger retention, and more meaningful progress visibility.

trainer workflows semantic annotation review and retention structured learning material
  • Organize training material through clearer learning workflows
  • Annotate positions with motifs and context, not only moves and comments
  • Reinforce learning over time through review tied to real material

Foundation layer

Mirasen Chessboard

Mirasen Chessboard is the open-source chessboard UI library inside the broader Mirasen ecosystem — a modern foundation for rendering, interaction, and richer chess applications.

Why it exists

The board is not the product story by itself. It is the foundation that supports richer learning workflows, review interactions, and future Mirasen tools.

How it fits

Chessboard supports the broader Mirasen learning ecosystem while remaining a distinct open-source library with its own public GitHub and npm presence.
